Single lipid component membrane bilayer MD with CHARMM36 force field, simulated with the CHARMM program

Data for ten single component lipid bilayer simulations, with 3 files per lipid: a DCD file with coordinates, a PSF file describing the system, and a .zip file containing the starting coordinate set (CHARMM COOR format) and the other inputs used for the CHARMM simulations.  Only the POPG system includes ions: Na+ to neutralize the lipids, and ca. 0.15 M NaCl.
The DCD trajectory files contain coordinate sets stored at 0.1 ns intervals, 
and are in the original CHARMM binary format.

Pure POPC Membrane Simulation Using Charmm-Drude Force Field with OpenMM

400 ns MD simulation of pure POPC membrane using Charmm-Drude polarizable force field. The system contains 72 POPC lipids and 2809 SWM4 water molecules.
The simulation have been performed using OpenMM 7.4.1
Before running the Drude simulation, the system has been equilibriated using Charmm36 force field for 200 ns. The last frame of that simulation was used to generate Drude polarizable model. The first 100 ns of the Drude simulation has been discarded from this dataset.
This dataset does not contain the water molecules.
Please note that the trajectories might need to be realigned.

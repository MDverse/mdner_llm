Simulations POPC bilayers (512 lipids) using charmm36 ff in gromacs
Collection simulations of POPC (512 lipids) bilayers in gromacs using the charmm36 force field. The list of systems can be found below where the several parameter are:

POPC_512_310K (500ns)
POPC_512_NaCl_150mM_310K (500ns)
POPC_512_NaCl_150mM_310K_tip3p (500ns)
POPC_512_NaCl_Dang_150mM_310K (500ns)

For further information read the Readme file provided for each simulation.

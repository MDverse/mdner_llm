Simulations of METTL3/METTL14 in complex with SAM+ADE or SAH+m6ADE

The current dataset contains a set of MD simulation trajectories for the METTLL3/METTL14 heterodimer in complex with the set of (co-)substrates or (co-)products SAM+ADE or SAH+m6ADE respectively. Each compressed file contains 16 independent trajectories of 500 ns each. The trajectories are written in .xtc format, and therefore a .pdb file is needed to read them. The parameters file ("md_input.mdp") is also provided. The current dataset also contains the PLUMED file that was used for the DFTB3/MM metadynamics simulations ("plumed.dat").

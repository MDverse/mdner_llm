Discriminating extracellular vesicles by their membranes


MD trajectories of the eight systems discussed in the paper are provided and can be visualized using molecular visualization programs such as Pymol or VMD uploading the GRO file and then the XTC.

Each folder contains the following files:

topology (.top);
trajectory files (.xtc) numbered progressively;
atomic coordinate (.gro);
index of the structure (.ndx);
the toppar folder;
portable binary run input file (.tpr).

For the system M1_310K the GRO file has a different name due to the fact that the simulation started from the last frame of the system M1_303.15K.

An example of input file used for the production step of the dynamics has been provided (Production.mdp).

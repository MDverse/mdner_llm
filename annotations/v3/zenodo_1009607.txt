Set simulations small pure bilayers (72 lipids) using charmm36 ff in gromacs (DPPC, POPC)
Collection simulations of DPPC and POPC bilayers in gromacs using the charmm36 force field. The list of systems describing their particular simulation conditions can be found below:
1) DPPC_72_325K (500ns)
2) DPPC_72_310K_rmcomm_leaflets (500ns)
3) DPPC_72_310K_rmcomm_leaflets_low_hydration (500ns)
4) POPC_72_310K (500ns)
5) POPC_72_310K_rmcomm_leaflets (500ns)
6) POPC_72_310K_rmcomm_leaflets_low_hydration (550ns)
7) POPC_72_303K_rmcomm_leaflets_low_hydration (550ns)
For further information read the Readme file provided for each simulation.

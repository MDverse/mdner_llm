ADMET
This folder includes five files, each containing the SMILES string used for ADMET analysis.
 
docking
This directory includes all the required files for docking. It contains four folders, 
    each representing a different statin derivative with its corresponding analogues.
 
MD Simulation
This folder contains three subfolders with the input files for MD simulations of the 
Apo protein, the Simvastatin complex, and the Lovastatin complex, respectively.
